  1. What is Semantic HTML? 
            HTML that has a name that represents what it does
    2. What is HTML used for? 
            Html is a markup language that communicates to a browser how to display the information
    3. What is an attribute and where do we put it? 
            an attribute includes information about an element, and does not render on a page. It is included inline after an element.
    4. What is the h1 tag used for? How many times should I use it on a page?
            An h1 tag is used to communicate a header; it is the largest header and should only be used once to communicate the most important information.
    5. Name two tags that have required attributes
        <a> <img>
    6. What do we put in the head of our HTML document? 
        the title of the webpage <title> and <style> if using an internal stylesheet.
    7. What is an id? 
        an identifier assigned to an element using # in css
    8. What elements can I add an id to? 
        all elements
    9. How many times can I use the same id on a page? 
        1
    10. What is a class? 
         a class is a selector assigned to an element using . 
    11. What elements can I add a class to? 
         any element
    12. How many times can I use the same class on a page?
         more than once 
    13. How do I get my link to open in a new tab?
         <a href = "link.html" target="_blank">
    14. What is the alt attribute in the image tag used for? 
         the alt attribute displays text (a desription of the image) if the image can not be displayed on a page
    15. How do I reference an id?
            #idName
    16. What is the difference between a section and a div
            section is a semantic tag; it does not affect the layout of a page. A div is a tag that actually puts the contents into sections
    17. What is CSS used for? 
            css is used to style html pages
    18. How to we select an element? Example - every h2 on the page

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
        an id has to be unique to one element, while classes can be assigned to multiple elements. an id could be used to identify an entire section of a page such as a comment section; 
        a class could be used to style a divs that need certain properties such as borders, font, etc.

            .className
    21. How do we select a p element with a single class of “human””?

    22. What is a parent child selector? When would this be useful? 
        a selector that is nested in another element. For example you would use a parent child selector to select all p elements in a div.
    23. How do you select all links within a div with the class of sidebar?

        }
    24. What is a pseudo selector?
            a selector that refers to elements in a specific state such as a hover state :hover
    25. What do we use the change the spacing between lines?

    26. What do we use to change the spacing between letters?
            letter-spacing
